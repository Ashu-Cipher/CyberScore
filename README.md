# CyberScore

## 📖 Description
* CyberScore is a web based assessment tool design to test and improve user awareness against modern cyner threat. It takes you through 40 real world scenarios, and scores you like credit score.

## 🛠️ How It Works
* The player (You) start with base score of 500. As you move forward and make choce, it affects your score, add point for right answer and deduct for wrong. The game groups questions into categories to see if you're good at these categories or need improvement. When you finish quiz, you get your score and also you get to know about your weakest spot so that you can be careful about it.

## 🛠️ How To Use?
* Go to [Site](https://ashu-cipher.github.io/CyberScore/), click on start assessment, complete quiz, know your points and weak spot to fix it.
